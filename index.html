jquery.xslt

A lightweight jquery plugin to provide XSLT support via the Sarissa javascript library.

Features

  • Simple, configurable caching of compiled stylesheets
  • Customisable jQuery.ajax integration for pulling server side stylesheets and content
  • Automates workarounds so that xsl:include, xsl:import and the document function work in browsers where support for these is inconsistent or just plain broken.
  • Support for custom URL resolvers
  • Return a string, DOM (Document) object or jQuery wrapped collection.

Planned Features

  • Sammy plugin/integration (xslt as a template engine)
  • Clean control over async/sync execution of transforms and ajax calls
  • Specify DOM elements to populate with results e.g., $('#main').transform(xsl, xml);
  • Transformation pipelines
  • An in-browser xproc implementation.
  • Assert compatibility with specific features (e.g., $('#target').transform({require: 'xsl:import', stylesheet: xsl}))

Cheat Sheet

Transforming local data.... 

$.xslt.transform({stylesheet: xslt, source: xml});

Transforming jQuery wrapped data.... 

$('#hidden data').xslt({ stylesheet: '/assets/xsl/render.xslt' });

Transforming server side data.... 

$.xslt.transform({
	stylesheetUrl: '/assets/xsl/convert2svg.xslt', 
	sourceUrl: '/resources/designs'
});

Caching Transforms.... 

$.xslt.transform({
	stylesheetUrl: '/assets/xsl/convert2svg.xslt', 
	sourceUrl: '/resources/designs',
	useCash: true
});

Getting the result as a DOM (Document) object.... 

$.xslt.transform({stylesheet: xslt, source: xml, resultFormat: 'DOM'});
